# AI-Trouble-Shooting-using-Voice-Assistance-
The primary goal of this project is to enable users to report or describe issues using voice input, allowing an AI agent to automatically analyze and resolve the problems without human intervention. This approach helps reduce response time, improve efficiency, and minimize the need for human resources.

Additionally, the system collects user feedback after issue resolution, which is used to continuously improve the agent’s performance and enhance its ability to handle future problems more effectively.
